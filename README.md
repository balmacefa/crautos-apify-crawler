# crautos-apify-crawler

The purpose of this crawler is to fetch all the data from the 'used car category' to send it later to an ELK stack for furder analysis.

- Depreciation
- Cost per km
- Buy recommendation

filter by:

- Automaker
- Model
- Year
- Motor CC
- ....
